Easily create a quick payment button. 

== Description ==
This will create a PayPal Standard button to receive payments online. It will automatically default to US dollars unless over-ridden in the shortcode. The client will need to specific the amount at PayPal. 

== Installation ==
1. Upload this plugin to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Place shortcode [tron_paypal_button_lite email="you@example.com" /] where you want the button.
